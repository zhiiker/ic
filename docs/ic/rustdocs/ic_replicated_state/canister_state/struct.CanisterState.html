pub struct CanisterState {
    pub system_state: SystemState,
    pub execution_state: Option<ExecutionState>,
    pub scheduler_state: SchedulerState,
}

The full state of a single canister.

Fields

system_state: SystemState

See SystemState for documentation.

execution_state: Option<ExecutionState>

See ExecutionState for documentation.

This may or may not exist depending on whether or not the canister has an actual wasm module. A valid canister is not required to contain a Wasm module. Canisters without Wasm modules can exist as a store of DFNs; temporarily when they are being upgraded, etc.
